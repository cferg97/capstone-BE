
# Capstone Backend

Link to front end documentation: https://github.com/cferg97/capstone-FE/blob/main/README.md

Using ExpressJS, MongoDB and Node.js

## Primary Endpoints Required

### For Users:
 - Register
 - Log In w/ Email + password
 - Log In w/ Google
 - Validate Account (email token)

### For Product Search (with pagination)
 - Get all products matching search query
    - And all matching variants
 - Get all products from chosen set
 - Show listings from chosen card along with card details

### For Seller Search (pagination)
 - Show all active sellers (users with active listings) matching search query

### Chat between users
 - Only one on one, basic messaging system
